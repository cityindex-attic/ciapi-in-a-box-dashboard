MetaData

The City Index API is self documented via JSON Schema and Service Mapping Description (SMD) meta data.

Combined, the JSON Schema and SMD meta data provide all relevant information regarding service methods and data types in a human and machine readable JSON format.

Using this meta data as reference, one can confidently author or generate code that accesses the City Index API.

An example in point is this application, the City Index API Explorer. It is a simple framework that consumes the City Index API JSON Schema and SMD meta data and dynamically renders documentation and sample javascript code.

Another example is the CIAPI.JS client library. It is based upon the Dojo JavaScript framework which provides built-in support for SMD.
